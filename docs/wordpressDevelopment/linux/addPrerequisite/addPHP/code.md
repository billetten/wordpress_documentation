#### Install PHP 7.0
```cmd
sudo apt update
```
```cmd
sudo apt install php7.0
```
```cmd
sudo apt install php7.0-common php7.0-mysql php7.0-xml php7.0-xmlrpc php7.0-curl php7.0-gd php7.0-imagick php7.0-cli php7.0-dev php7.0-imap php7.0-mbstring php7.0-opcache php7.0-soap php7.0-zip php7.0-intl php7.0-fpm -y
```
